# 文字合成ツール (Character Merger Tool)

2つの文字を合成・加工して、新しいロゴや造形を作成するためのシングルページWebアプリケーションです。

## 主な機能

- **ツイン・キャンバス構成**: 合成結果を確認するプレビューキャンバスと、部品を加工する編集キャンバスを並べて操作。
- **テキスト入力・フォント設定**: PC内のローカルフォントを読み込み、自由に選択可能。
- **消しゴム＆やり直し (Undo)**: 部品文字をマウスでなぞって消去でき、Undo機能でやり直しが可能。
- **配置調整**: 1px単位の座標指定や矢印ボタン、長押しによる連続移動での精密な位置合わせ。
- **合成エクスポート**: 重なりを確認しながら、PNG（透過）またはBMP形式で保存。保存先の自由な指定が可能。

## 技術スタック

- **Frontend**: React, Vite
- **Canvas API**: 高度なピクセル操作と合成処理
- **File System Access API**: ローカルPCへの自由な画像保存

## 開発環境の実行

1. `npm install` (依存関係のインストール)
2. `npm run dev` (開発サーバーの起動)
3. ブラウザで `http://localhost:5173` を開く
